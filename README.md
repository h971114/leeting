## ✨ 주요 기능

### Leeting

> 사용자들 간에 Lan선을 통한 Meeting

- Leeting
  - 원하는 모임이 있는 경우 Leeting 등록
  - 썸네일과 부제목을 통한 유저 흥미도 유도
  -  참여한 모임에 따른 추천, 현재 인기있는 모임 추천
  -  웹엑스링크를 통하여 화상대화 편의성 제공
  - 어뷰징 시스템
    - Leeting 내 이상 유저의 경우 신고 가능
    - 여러번 신고 당했을 경우 계정 정지 가능
  - 리뷰
    - Leeting을 참여한 사람에 의해서 리뷰 작성 가능
